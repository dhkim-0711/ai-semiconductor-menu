# 엑셀형 편집 파일 안내

`data/projects-master.csv`
- 엑셀에서 바로 열어 수정할 수 있는 관리용 파일입니다.
- 각 행은 하나의 사업입니다.
- 여러 문장은 ` || ` 로 구분되어 있습니다.
- 링크는 `제목 >>> URL >>> 설명` 형식으로 들어 있습니다.

`data/projects-master.xls`
- Excel에서 바로 열 수 있는 호환 편집 파일입니다.
- 보기나 공유용으로 편하게 쓸 수 있습니다.
- 수정 후 웹 반영까지 하려면 CSV에도 같은 내용을 저장해 주세요.

웹 반영 방법
1. `data/projects-master.csv`를 수정합니다.
2. 프로젝트 폴더에서 `웹데이터_반영.bat`를 실행합니다.
3. `data/portfolio-data.generated.js`와 `data/projects-master.xls`가 함께 갱신됩니다.

웹이 읽는 데이터 파일
- `data/portfolio-data.generated.js`

참고
- 웹 화면은 이제 `app.js`와 `data/portfolio-data.generated.js`를 기준으로 동작합니다.
